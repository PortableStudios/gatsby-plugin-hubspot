# gatsby-plugin-hubspot

A Gatsby plugin to easily add a HubSpot embed code to your site.

## Installing

`npm install --save gatsby-plugin-hubspot`

## How to use

```js
// In your gatsby-config.js
plugins: [
  {
    resolve: `gatsby-plugin-hubspot`,
    options: {
        trackingCode: 1234567
    },
  },
]
```
